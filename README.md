🚗 Plate Recognition App (Android)

Приложение для распознавания автомобильных номеров в реальном времени с использованием камеры устройства.

📥 Установка
Перейдите в раздел Releases репозитория
Скачайте файл .apk
Установите APK на устройство Android
При установке разрешите установку из неизвестных источников (если требуется)
▶️ Запуск
Откройте приложение
Разрешите доступ к камере
Наведите камеру на автомобильный номер
⚙️ Как работает приложение

Приложение выполняет следующие шаги:

Камера → Кадр → OCR → Поиск номера → Вывод результата
используется камера устройства (CameraX)
текст распознаётся через ML Kit OCR
номер определяется с помощью регулярного выражения
🇷🇺 Поддерживаемый формат номеров

Приложение распознаёт номера формата:

A000AA 101

Где:

1 буква
3 цифры
2 буквы
регион (2–3 цифры)
Примеры:
A123BC 77
O456AA 123
M999KK 78
📱 Интерфейс
📷 Камера в реальном времени
📝 Отображение распознанного номера
🔔 Всплывающее окно с результатом
⚠️ Особенности работы
распознавание происходит в реальном времени
окно с номером появляется только один раз (до нажатия OK)
одинаковые номера не повторяются слишком часто
❗ Ограничения

Приложение использует OCR, поэтому:

может распознавать лишний текст
возможны ошибки (например O ↔ 0)
не всегда точно выделяет номер
